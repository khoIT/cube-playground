## No-op Strategy

`createLangfuseClient()` in `langfuse-client.ts` returns `null` when:
- `LANGFUSE_PUBLIC_KEY` or `LANGFUSE_SECRET_KEY` absent (checked via `isLangfuseEnabled()`)
- SDK constructor throws (bad host URL etc.) — caught and warned

`LangfuseTracer.disabled = (client === null)`. Every `ObserverHooks` method and `finalize()` guard with `if (this.disabled) return` as first line — zero allocations on disabled path.

Verified: `disabled: true`, `client: null` when keys unset. All methods execute without throw or Langfuse network call.
