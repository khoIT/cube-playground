## Langfuse Deep-Link Detection

Read at module init in `turn-detail.tsx`:
```ts
const LANGFUSE_HOST = (import.meta as Record<string, any>).env?.VITE_LANGFUSE_HOST as string | undefined;
```
When set: renders `<a href="${LANGFUSE_HOST}/trace/${turn.id}">Open in Langfuse</a>` in the assistant-turn header. When unset (default): link hidden. `VITE_LANGFUSE_HOST` added to `.env.example` as a commented line with format guidance.
