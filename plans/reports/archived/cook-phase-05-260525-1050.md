## FE SSE Event Types — Unchanged

Before and after, `emit()` is called for exactly these event types (listed in order of possible emission):

1. `session_created` — only if new session
2. `compact_warning` — only if auto-compact ran
3. `turn_started`
4. `loading`
5. `query_artifact` — via sseEmitter, zero or more
6. `chart` — via sseEmitter, zero or more
7. All events from `claudeRunner.run()` that are not `query_artifact`/`chart` (e.g. `token`, `tool_call`, `tool_result`, `thinking`, `result`)
8. `done` (success path) or `error` (error path)

Observer events flow through a side-channel only — they never touch `emit()` or the SSE wire.
