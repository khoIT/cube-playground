NewSegmentsJune 14, 2026

Lakehouse Snapshot Inbox

The nightly segment-membership snapshot now has a live operational view in the Admin hub.

Admin · Segment refreshes · Snapshots

What's included:

  • Per-segment breakdown — which segments landed, row counts, skips and errors.
  • A live-run indicator that streams segments in as they land.
  • Owner provenance so you can see whose segment each row belongs to.
  • Run-level error surfacing when the lakehouse is unreachable.
